## ✨ НОВЫЕ ФУНКЦИИ:

### 📱 Экран информации о чате (как в Telegram)

**Как открыть:**
Нажмите на иконку **ℹ️** в правом верхнем углу чата

**Вкладки:**

1. **🔍 Поиск** - Поиск сообщений по чату
   - Введите текст для поиска
   - Показывает все совпадения
   - Клик по результату → переход к сообщению

2. **🖼️ Медиа** - Все фото и видео
   - Сетка 3x3
   - Клик на фото → открывает в новой вкладке
   - Видео с иконкой 🎬

3. **📄 Файлы** - Все документы
   - Список файлов с названием
   - Показывает дату и размер
   - Клик → скачивание/открытие

4. **🎤 Голосовые** - Все голосовые сообщения
   - Список с именем отправителя
   - Показывает дату
   - Клик → воспроизведение

5. **🔗 Ссылки** - Все ссылки из сообщений
   - Список всех URL
   - Показывает кто отправил и когда
   - Клик → открывает ссылку
