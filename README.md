# 💬 Polka - Мессенджер как Telegram

Современный мессенджер с поддержкой текстовых сообщений, медиа, голосовых и видео сообщений, звонков.

## ✨ Функции

### Основные:
- ✅ Текстовые сообщения в реальном времени
- ✅ Персональные и групповые чаты
- ✅ Загрузка медиа (фото, видео, файлы)
- ✅ Голосовые сообщения
- ✅ Видео-кружочки (как в Telegram)
- ✅ Аудио и видео звонки (WebRTC)

### Дополнительно:
- ✅ Ответы на сообщения (Reply)
- ✅ Редактирование сообщений (48ч)
- ✅ Удаление сообщений (для себя/для всех)
- ✅ Эмодзи реакции
- ✅ Превью ссылок
- ✅ Поиск по чатам и сообщениям
- ✅ Темная тема
- ✅ Статусы онлайн/офлайн
- ✅ Индикатор "печатает..."
- ✅ Статусы доставки (галочки)
- ✅ Управление группами
- ✅ Профили с аватарами

## 🛠️ Технологии

### Backend:
- Node.js + Express
- Socket.io (WebSocket)
- PostgreSQL + Sequelize
- Redis (опционально)
- WebRTC (для звонков)
- JWT аутентификация

### Frontend:
- React Native (Expo)
- React Native Web
- Socket.io client
- Axios
- AsyncStorage

## 📦 Установка и запуск

### Требования:
- Node.js 16+
- PostgreSQL 14+
- Redis (опционально)

### Backend:

```bash
cd backend
npm install
cp .env.example .env
# Отредактируйте .env - укажите данные PostgreSQL
node src/config/initDatabase.js
npm run migrate
npm start
```

### Frontend:

```bash
cd mobile
npm install
npx expo start --web
```

## 🌐 Деплой в Production

Подробная инструкция: `DEPLOYMENT_GUIDE.md`

**Быстрый деплой:**
1. **Backend** → Railway.app (с PostgreSQL)
2. **Frontend** → Vercel.com
3. **Файлы** → Cloudinary

## 📱 Поддерживаемые платформы

- ✅ **Web** (полная поддержка)
- ⚠️ **iOS** (требует доработка)
- ⚠️ **Android** (требует доработка)

**Примечание:** Голосовые, видео-кружочки и звонки пока только на Web.
