# Кружочки - звук при клике

## ✅ ЧТО ИСПРАВЛЕНО:

### 1. **Клик по кружочку → воспроизведение со звуком**
- **По умолчанию:** видео зациклено, БЕЗ звука (muted)
- **При клике:** 
  - Перематывается на начало (`currentTime = 0`)
  - Убирается зацикливание (`loop = false`)
  - **Включается звук** (`muted = false`)
  - Воспроизводится один раз
- **После окончания:** возвращается зацикливание и без звука

### 2. **Убраны границы**
- Добавлено `border: 'none'` и `outline: 'none'` к видео
- Добавлено `borderWidth: 0` к контейнеру
- Добавлено `display: 'block'` для устранения inline-gaps

## 🎯 КАК РАБОТАЕТ:

### По умолчанию:
- ✅ Видео воспроизводится автоматически
- ✅ Зацикленное (`loop: true`)
- ✅ БЕЗ звука (`muted: true`)
- ✅ Тихо крутится в фоне

### При клике:
```javascript
onClick={(e) => {
  const video = e.target;
  video.currentTime = 0;  // Перематываем на начало
  video.loop = false;      // Убираем зацикливание
  video.muted = false;     // ВКЛЮЧАЕМ ЗВУК
  video.play();            // Воспроизводим
}}
```

### После окончания:
```javascript
onEnded={(e) => {
  const video = e.target;
  video.loop = true;   // Возвращаем зацикливание
  video.muted = true;  // Возвращаем без звука
  video.play();        // Продолжаем воспроизведение
}}
```
