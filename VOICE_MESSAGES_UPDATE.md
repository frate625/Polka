## 📁 СОЗДАННЫЕ/ИЗМЕНЕННЫЕ ФАЙЛЫ:

### 1. **`mobile/src/components/VoicePicker.js`** (НОВЫЙ)
Компонент для записи голосовых сообщений, аналогичный `VideoNotePicker`:

**Функции:**
- `startRecording()` - запрашивает доступ к микрофону и начинает запись
- `stopRecording()` - останавливает запись и отправляет
- `cancelRecording()` - отменяет запись (не отправляет)
- `uploadVoice()` - загружает аудио файл на сервер

**Особенности:**
- Использует `MediaRecorder` API (Web)
- Формат: `audio/webm;codecs=opus`
- Максимальная длительность: 5 минут
- Флаг `isCancelledRef` для отмены

## 💡 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### MediaRecorder API:
- **Формат:** `audio/webm` с кодеком `opus`
- **Макс. длина:** 5 минут (300 секунд)
- **Контроль:** Полный контроль через JS

### Отмена записи:
```javascript
isCancelledRef.current = true; // устанавливаем ДО .stop()
mediaRecorder.stop(); // триггерит onstop
// в onstop проверяем флаг и не загружаем
```
