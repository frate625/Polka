# Звуки для уведомлений

## Как добавить свой звук:

1. Скачайте звуковой файл (.mp3, .wav, .ogg)
2. Переименуйте в `notification.mp3`
3. Положите сюда: `mobile/assets/sounds/notification.mp3`
4. Перезапустите приложение

## Рекомендации:

- **Формат:** MP3 (лучшая совместимость)
- **Длина:** 0.5-2 секунды (короткий звук)
- **Громкость:** Нормализованная (не слишком громко)
- **Битрейт:** 128 kbps (достаточно)

## Где найти звуки:

- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/ (тег: notification)
