Чтобы кнопка "Выйти" заработала на https://polka-pi.vercel.app

#### Если у вас есть Git репозиторий:

```bash
cd C:\Users\Gamer\chat-app

# Проверить статус
git status

# Добавить изменения
git add mobile/src/screens/ProfileScreen.js
git add mobile/src/store/AuthContext.js

# Закоммитить
git commit -m "Fix logout button for Web platform"

# Отправить на GitHub
git push
```

После push, Vercel **автоматически задеплоит** изменения (~2-3 минуты).

#### Если НЕТ Git репозитория:

**Вариант 1: Создать Git репозиторий**

```bash
cd C:\Users\Gamer\chat-app

# Инициализировать git
git init

# Добавить файлы
git add .

# Первый коммит
git commit -m "Initial commit with logout fix"

# Создать репозиторий на GitHub:
# 1. Зайти на https://github.com/new
# 2. Создать новый репозиторий "polka-messenger"
# 3. Выполнить команды которые GitHub покажет:

git remote add origin https://github.com/ВАШ_USERNAME/polka-messenger.git
git branch -M main
git push -u origin main
```

Затем в Vercel:
1. Зайти в проект
2. Settings → Git → Connect Repository
3. Выбрать ваш новый репозиторий

**Вариант 2: Ручной деплой через Vercel CLI**

```bash
# Установить Vercel CLI (если нет)
npm install -g vercel

# Войти
vercel login

# Перейти в папку mobile
cd mobile

# Задеплоить
vercel --prod
```

## 🔍 КАК ПРОВЕРИТЬ НА PRODUCTION

После деплоя на Vercel:

1. Откройте: https://polka-pi.vercel.app
2. Откройте консоль браузера (F12 → Console)
3. Войдите в аккаунт
4. Перейдите в "Профиль"
5. Нажмите "Выйти"
6. Подтвердите в браузерном окне
7. В консоли должны быть логи:
   ```
   🚪 Начинается выход из аккаунта...
   🚪 Logout: начало процесса выхода
   ✅ Logout: API logout успешно
   ✅ Logout: завершено успешно
   ```
8. ✅ Должно вернуть на экран входа

## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Backend не запускается:
```bash
# Проверить PostgreSQL
services.msc → найти postgresql → Start

# Проверить .env файл
notepad backend\.env
# Убедиться что DATABASE_URL правильный
```

### Frontend не подключается к Backend:
```bash
# Проверить что Backend запущен
curl http://localhost:3000
# Должен вернуть JSON

# Проверить .env Frontend
notepad mobile\.env
```

### Git не работает:
```bash
# Установить Git для Windows
# https://git-scm.com/download/win

# Проверить установку
git --version
```

### Vercel деплой упал:
1. Зайти в Vercel Dashboard
2. Deployments → выбрать последний
3. View Function Logs
4. Посмотреть ошибку
