## 📱 КАК ПОЛЬЗОВАТЬСЯ

### 1. Регистрация/Вход

1. Откройте: https://polka-pi.vercel.app
2. Заполните форму регистрации:
   - Имя пользователя
   - Email
   - Пароль
   - Телефон (опционально)
3. Нажмите "Зарегистрироваться"

Или войдите, если уже есть аккаунт:
- Email
- Пароль
- Нажмите "Войти"

### 2. Создание чата

**Личный чат:**
1. На вкладке "Чаты" → кнопка "+"
2. Выберите пользователя из списка
3. Начните общение

**Групповой чат:**
1. "Чаты" → "+" → "Создать группу"
2. Введите название группы
3. Выберите участников
4. Нажмите "Создать"

### 3. Отправка сообщений

**Текст:**
- Введите текст в поле внизу
- Нажмите "Отправить" или Enter

**Фото/Видео:**
- Нажмите на иконку 📎 (скрепка)
- Выберите файл
- Отправить

**Голосовое:**
- Нажмите и удерживайте кнопку 🎤
- Говорите
- Отпустите для отправки
- Свайп влево для отмены

**Видео-кружочек:**
- Нажмите и удерживайте кнопку 📹
- Держите для записи (до 60 сек)
- Отпустите для отправки
- Свайп влево для отмены

### 4. Действия с сообщениями

**Ответить:**
- Долгое нажатие на сообщение (или свайп вправо)
- Выберите "Ответить"
- Введите ответ

**Редактировать:**
- Долгое нажатие на СВОЁ сообщение
- "Редактировать"
- Измените текст
- Сохранить

**Удалить:**
- Долгое нажатие на сообщение
- "Удалить"
- Выберите: "Для себя" или "Для всех"

**Реакция:**
- Долгое нажатие
- Выберите эмодзи

### 5. Настройки профиля

1. Перейдите на вкладку "Профиль"
2. Нажмите "Редактировать профиль"
3. Измените:
   - Имя пользователя
   - Статус
   - Аватар (нажмите на фото)
4. Сохраните

**Смена темы:**
- Профиль → Тема
- Выберите: Светлая / Темная / Авто

### 6. Выход

1. Профиль → кнопка "Выйти" (внизу, красная)
2. Подтвердите выход
3. ✅ Готово!

## 🔧 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### Сообщения не отправляются
**Причина:** Backend не доступен или проблема с сетью  
**Решение:** Проверьте подключение к интернету, обновите страницу

### Файлы не загружаются
**Причина:** Проблема с хранилищем или размер файла > 50MB  
**Решение:** Проверьте размер файла, попробуйте другой файл

### "Печатает..." не появляется
**Причина:** WebSocket не подключен  
**Решение:** Обновите страницу, проверьте консоль (F12)

### Голосовые/видео-кружочки не работают
**Причина:** Нет разрешения на микрофон/камеру  
**Решение:** Разрешите доступ в браузере

### Не работает выход
**Причина:** Была исправлена в последнем обновлении  
**Решение:** 
- Local: Обновите код
- Vercel: Должно работать сейчас (https://polka-pi.vercel.app)
